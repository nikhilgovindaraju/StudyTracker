# Study Tracker

A personal interview preparation tracker built with React.

This tool helps track progress across multiple interview preparation topics including:

- Data Structures & Algorithms
- System Design
- SQL / Databases
- Behavioral Preparation
- Daily Study Tasks
- Study Sessions with Timer

The application allows you to manage topics, track daily study goals, log sessions, and analyze progress over time.

## Features

### Topic Tracker
Organize interview preparation topics into sections such as:

- DSA
- System Design
- SQL
- Behavioral

Mark topics to add them to today's study queue.

### Daily Study Queue

View all topics scheduled for today and mark them as completed.

### Study Timer

Start study sessions with or without selecting a specific topic.

Sessions are saved and contribute to study analytics.

### Analytics Dashboard

The analytics section provides:

- Total study time
- Study streak
- Daily study breakdown
- Section completion progress
- Weekly study overview

### Persistent Storage

All progress is stored locally using browser `localStorage`, so your progress persists across refreshes.
